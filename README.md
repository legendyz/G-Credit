# G-Credit

BMAD (Business Model Agent Development) 框架项目

## 项目简介

这是一个基于Agent的开发框架，包含多个专业领域的智能助手和工作流。

## 项目结构

- `_bmad/` - BMAD框架核心配置和模块
  - `_config/` - 配置文件和清单
  - `_memory/` - 记忆和状态管理
  - `bmb/` - 构建器模块
  - `bmgd/` - 游戏开发模块
  - `bmm/` - 主模块
  - `cis/` - 创新策略模块
  - `core/` - 核心功能模块
- `_bmad-output/` - 输出产物目录
- `docs/` - 文档目录

## 快速开始

```bash
# 克隆项目
git clone https://github.com/YOUR_USERNAME/G-Credit.git

# 进入项目目录
cd G-Credit
```
