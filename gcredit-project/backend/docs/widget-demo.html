Overview

The G-Credit Badge Widget allows you to embed your digital badges on any website. This page demonstrates different widget configurations and provides embedding instructions.

API Endpoints

Get Badge Data (JSON):
GET /api/badges/{badgeId}/embed
Get Widget HTML:
GET /api/badges/{badgeId}/widget?size=medium&theme=light&showDetails=true

Query Parameters

  • size: small, medium, large (default: medium)
  • theme: light, dark, auto (default: light)
  • showDetails: true, false (default: true)

Embedding Methods

Method 1: Direct HTML (Recommended)

Fetch the widget HTML from the API and inject it into your page:

<div id="badge-widget"></div>

<script>
async function loadBadgeWidget(badgeId, containerId, options = {}) {
  const { size = 'medium', theme = 'light', showDetails = true } = options;
  const apiUrl = `http://localhost:3000/api/badges/${badgeId}/widget`;
  const params = new URLSearchParams({ size, theme, showDetails: String(showDetails) });
  
  try {
    const response = await fetch(`${apiUrl}?${params}`);
    const widget = await response.json();
    
    const container = document.getElementById(containerId);
    
    // Inject CSS
    const style = document.createElement('style');
    style.textContent = widget.css;
    document.head.appendChild(style);
    
    // Inject HTML
    container.innerHTML = widget.html;
    
    // Inject JavaScript
    const script = document.createElement('script');
    script.textContent = widget.script;
    document.body.appendChild(script);
  } catch (error) {
    console.error('Failed to load badge widget:', error);
  }
}

// Load widget
loadBadgeWidget('YOUR_BADGE_ID', 'badge-widget', {
  size: 'medium',
  theme: 'light',
  showDetails: true
});
</script>

Method 2: Iframe Embedding

Create a dedicated page that renders the widget and embed it via iframe:

<iframe
  src="http://localhost:3000/widget.html?badgeId=YOUR_BADGE_ID&size=medium&theme=light"
  width="220"
  height="320"
  frameborder="0"
  style="border: none;"
></iframe>
Note: For iframe embedding, you'll need to create a dedicated widget page that calls the API.

Method 3: JSON API (Custom Rendering)

Fetch badge data and render it yourself:

fetch('http://localhost:3000/api/badges/YOUR_BADGE_ID/embed')
  .then(response => response.json())
  .then(badge => {
    // Render badge with your own HTML/CSS
    console.log(badge.badgeName);
    console.log(badge.badgeImageUrl);
    console.log(badge.verificationUrl);
  });
